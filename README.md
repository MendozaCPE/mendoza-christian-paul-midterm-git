`Midterm Examination Reflection`

Answer the following questions briefly:

1. What is Git used for?
   Git is a distributed version control system used to track changes in source code. It allows developers to monitor modifications, revert to previous versions, and collaborate on files over time.
2. What is the difference between Git and GitHub?
   Git acts as a local software tool for managing code version control on your computer. In contrast, GitHub provides a cloud-based hosting platform built to store, share, and collaborate on those Git repositories.
3. Why is branching useful?
   Branching in software development allows you to isolate work, enabling parallel development. It help developer too for experenting on their project with easy control on the project version
4. What command did you use to upload your local commits to GitHub?
   git add .
   git commit -m"Initial Commit"
   git push -u origin master
5. What did you learn from this activity?
   Git provides a complete safety net by tracking your project's history so you can always rollback mistakes. It also serves as a detailed progress journal while helping you build a professional portfolio for future employers.
